# Model Files

This folder contains TF.js model files generated by train_model/train.py

Files that belong here (not committed to git):
- model.json
- weights.bin
- label_encoder.json
- scaler.json

To generate these files:
    cd train_model
    python -m venv venv
    venv\Scripts\activate
    pip install -r requirements.txt
    python train.py

Until model is trained, the app runs in mock detection mode.
Mock mode randomly shows signs every 3 seconds for UI testing.
